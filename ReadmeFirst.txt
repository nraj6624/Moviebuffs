It is a node js application.



Steps to run:
1. run api server 'server.js' as "node server.js" in command line.
2. run site server 'siteserver.js' in site folder at root.
3. open url localhost:8082 in your browser.
4. search any movie.
5. select any movie and now you can mark as watched.
6. Every time you search the movie you will find the movies as watched for the movies you already watched.
